# Bayesian-Portfolio-Optimisation

This repository implements Bayesian information updating for portfolio asset allocation, building on the Black-Litterman model framework. I explicitly construct the covariance matrix and demonstrate how investor views (new information) update prior return estimates via Bayes' theorem. The general workflow followed is outlined below:

1. Setup & Data Simulation
2. Explicit Covariance Matrix Construction
3. Prior Distribution (Equilibrium Returns)
4. Investor Views (Likelihood)
5. Bayesian Update (Posterior Returns)
6. Optimal Portfolio Weights
7. Visualisations & Comparison
